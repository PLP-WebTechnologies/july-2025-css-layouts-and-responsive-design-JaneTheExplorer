Layouts and grid

Flexbox Layout

Flexbox is used for the navigation, cards, and footer sections. It provides efficient alignment and distribution of space among items.

CSS Grid Layout

CSS Grid is used for the main page structure, creating a two-dimensional layout system with rows and columns.

Responsive Design

Media queries adjust the layout at different breakpoints, ensuring optimal viewing on mobile, tablet, and desktop devices.
